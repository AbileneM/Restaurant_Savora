    Comment démarrer le site
````
Pour démarrer le site de Savora
1. Faire 'npm start' dans le terminal
2. Copier 'http://localhost:5000/' dans votre barre de navigateur
3. Lancer dans le teminal 'npx @tailwindcss/cli -i ./public/styles/input.css -o ./public/styles/style.css --watch'. Laisse ce terminal ouvert.
4. Ensuite recharge ta page.

````
